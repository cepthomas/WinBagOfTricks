# WinBagOfTricks

====================== Going to sleep ======================
=========== Parts have migrated to other projects ==========

A place to play with some Windows stuff. Requires VS2022 and .NET8.


![logo](felix63.png)

- Tray application demo. Still works and could be useful.
- Clipboard with multiple entries. Like Win10/11 clipboard but some extra features.
- A simple taskbar/jumplist manager. Still works and could be useful.
- General purpose Win32 API library.
- Clipboard support for Console applications.
- Generic test files that are used by various external projects.
